GIT software de control de versiones:

controla los cambios realizado sobre un archivo o conjunto de los mismos
a lo largo del tiempo.
-gestiona distintas versiones de un mismo archivo

Repositorio:
lugar donde podemos guardar cosas
tipos:
locales: lo que se crean en la pc, no nesesariamente son compartidos
Remotos: no se encuentran en la pc, se encuentra en un servidor externo

1). Pasos para crear el primer repositorio:
.Descargamos git o instalamos en linux
.una vez instalada se habre la consola

-con cd ubicamos la carpeta de trabajo
-inicializar: git init
-git init
.agrega un subdirectorio oculto en ese subdirectorio guarda toda la informacion necesaria para que 
git comiense arastrear los cambios, historial configuraciones, ramas, etc.
.desde ese momento se puede comenzar a usar los comandos git add, git commit, etc
para registrar y controlar las versiones de tus archivos

.en pocas palabras con despues de ecribir  git init, la carpta se convierte en un repositorio de git
listo para llevar un historial de cambios

2). Abrir la terminal y escribo git init y por pantalla aparece: 
Salida: "Initialized empty Git repository in (ubicacion)"
o
si ya estaba inicializada(ya fue creada) tiene el mensaje
"Reinicializado el repositorio Git existente en (ubicacion)"

CREANDO UNA CUENTA EN EL REPOSITORIO REMOTO DE GIT

1.creamos una cuenta en github
https://github.com
que es github: es una plataforma en la nube donde los programadores guardan,
comparten y trabajan juntos en preoyectos de codigo. Funciona como un repositorio central
que permite colaborar, revisar cambios y mejorar el software de manera organizada y segura

Concetos Basicos: 
.GITHUB portal para el codigo, sitio web que aloja proyectos de programacion
.BASADO EN GIT: utilizagot, sistem de control de versiones que registra cada cambio del codigo
.Colaboracion: permite que varios desarrolladores trabajen en un mismo proyecto
.Repositorios: Los proyectos que se guardan en "repositorios", que puden se publicos
(abierto para todos) o privados (Solo para el equipo)

FUNCIONES PRINCIPALES:
.guardar el codigo
.control de versiones 
.revision de codigo
.colaboracion global
.opciones de privacidad
