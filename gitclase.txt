Si quiero creo una carpeta para trabajar, creamos una carpeta donde vamos a trabajar


Creamos un usuario desde la terminal

* git config --global user.name "usuarioGit"

creamos un correo electronico

* git config --global user.email "correoelectronico@dominio.com"

donde vemos los datos, desde la terminal buscamos el archivo oculto

* .gitconfig

en nuestro caso ls -a para ver los achivo ocultos...

Como indicamos que vamos a trabajar o inicializar nuestro proyecto usamos 

* git init
